======== ===============
airlines R Documentation

Airline names.
--------------

Description
~~~~~~~~~~~

Look up airline names from their carrier codes.

Usage
~~~~~

::

   airlines

Format
~~~~~~

Data frame with columns

carrier
   Two letter abbreviation.

name
   Full name.

Source
~~~~~~

https://www.transtats.bts.gov/DL_SelectFields.asp?Table_ID=236

Examples
~~~~~~~~

::

     airlines
